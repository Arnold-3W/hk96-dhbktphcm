# Lớp HK96 — Khoa Kỹ Thuật Hàng Không, ĐHBK TP.HCM

Trang landing page lưu niệm tình bạn hơn 30 năm của tập thể lớp **HK96** —
Khoa Kỹ Thuật Hàng Không, Đại học Bách Khoa TP.HCM (1996 – 2001).

## Nội dung
- Câu chuyện tình bạn và hành trình sau tốt nghiệp
- Danh sách thành viên
- Thư viện ảnh các mùa họp mặt (2022 – 2026) và kỷ niệm xưa

## Công nghệ
Trang tĩnh thuần HTML/CSS/JavaScript (một file `index.html`), không cần build.

## Chạy thử
Mở trực tiếp `index.html` bằng trình duyệt, hoặc deploy lên bất kỳ hosting tĩnh nào (Vercel, Netlify, GitHub Pages...).
